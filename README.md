Este  projeto  implementa  um  pipeline  completo  de  Deep  Learning  para  classificação  de imagens usando a biblioteca PyTorch. O objetivo central é construir e treinar uma Rede Neural Convolucional (CNN), chamada ConvNet,  para  classificar  corretamente  imagens  do  dataset  CIFAR-10,  que  é  composto  por  10 categorias (como "avião", "carro", "pássaro", etc.). 
Fonte dos dados:https://www.cs.toronto.edu/~kriz/cifar.html
O  script  inicia  configurando  o  ambiente,  importando  as  bibliotecas  necessárias  e,  de forma  inteligente,  seleciona  o  dispositivo  de  hardware  mais  potente  disponível  (GPU  NVIDIA, GPU Apple ou CPU) para acelerar o treinamento. Em seguida, ele carrega e prepara os dados de treino  e  teste,  aplicando  transformações  essenciais  que  convertem  as  imagens  em  Tensores  e normalizam  seus  valores  de  pixel  para  o  intervalo  [-1.0,  1.0],  um  passo  essencial  para  a estabilidade do treinamento.
A arquitetura  do  modelo  ConvNet  é  definida  com  duas  camadas  convolucionais  (para extração   de   padrões   visuais)   intercaladas   com  camadas   de   max-pooling   (para   reduzir   a dimensão),  seguidas  por  três  camadas  lineares  (totalmente  conectadas)  que  atuam  como  o "cérebro" classificador para tomar a decisão final. 
O  coração  do  projeto  é  o  loop  de  treinamento,  que  executa  por  10  épocas.  Dentro  de cada época, o script alimenta o modelo com os dados em lotes (batch_size = 64), realiza o forward pass  (previsão),  calcula  o  erro  (com  CrossEntropyLoss)  e,  através  do  backward  pass  e  do otimizador Adam, ajusta os pesos da rede para minimizar esse erro. A acurácia é verificada no conjunto de teste ao final de cada época para monitorar o progresso.Após  o  término  do  treinamento,  o  script  executa  uma  avaliação  final  e  detalhada  no conjunto   de  teste,  calculando   não  apenas  a  acurácia  geral  do  modelo,  mas  também  a performance específica para cada uma das 10 classes. O modelo treinado é então salvo em disco (.pth). A  última  seção  demonstra  o  "deploy"  e  uso  prático:  Carregamos  o  modelo  salvo  e  o utilizamos para classificar imagens locais (externas ao dataset), exibindo a imagem e a previsão da  IA  com  seu  respectivo  percentual  de  confiança.  Testamos  ainda  o  que  acontece  quando usamos  imagem  de  uma  classe  que  não  foi  usada  no  treinamento  do  modelo.  Concluímos  o projeto comparando o tempo de execução entre diferentes tipos de hardware.
